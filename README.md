# Домашнее задание №6 к курсу "Знакомство с языками программирования"

### В данном репозитории 4 папки. Каждая папка соответствует номеру задания.

1. Task1 - Показать двумерный массив размером m×n заполненный вещественными числами.
2. Task2 - Задать двумерный массив следующим правилом: Aₘₙ = m+n.
3. Task3 - В двумерном массиве заменить элементы, у которых оба индекса чётные на их квадраты.
4. Task4 - В матрице чисел найти сумму элементов главной диагонали.
